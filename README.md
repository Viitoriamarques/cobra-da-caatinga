# Cobra da Caatinga

Jogo desenvolvido como projeto educativo para o Museu Vivo Répteis da Caatinga,
localizado em Puxinanã, região metropolitana de Campina Grande - PB.

## Objetivo

Criar uma experiência lúdica para crianças visitantes do museu, ambientada no
bioma da Caatinga. O jogador controla uma cobra que deve coletar sapinhos
enquanto desvia dos gaviões pelo cenário.

## Como jogar

- Use as setas do teclado para mover a cobra
- Colete todos os sapinhos para vencer
- Desvie dos gaviões — cada colisão custa uma vida
- Você tem 3 vidas
- Pressione R para reiniciar após game over

## Tecnologias

- Python 3
- Pygame

## Como executar

1. Instale o Pygame: `pip install pygame`
2. Execute: `python jogo.py`
